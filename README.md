

개인 브랜치 생성

```bash
git branch {브랜치 이름}
```

개인 브랜치로 이동

```bash
git switch {브랜치 이름}
```



브랜치에 업로드

```bash
multicampus@DESKTOP-KVCQHCD MINGW64 ~/Desktop/S05P12C205/ ({브랜치 이름})      
$git add .
$git commit -m "커밋 메시지"
$git push origin {브랜치 이름}
```



브랜치 이동

```bash
$git switch master
```

머지

```bash
$git merge origin {브랜치 이름}
```

푸시

```bash
$git push origin master

```
